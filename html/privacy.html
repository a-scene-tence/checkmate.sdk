1 수집하는 개인정보

체크메이트는 Google 계정을 통한 소셜 로그인 방식을 사용합니다. 로그인 과정에서 아래 정보가 수집됩니다.

  • 이름 — 앱 상단 유저바 표시용
  • 이메일 주소 — 계정 식별 및 데이터 연결용
  • 프로필 사진 — 앱 상단 아바타 표시용 (없을 경우 이니셜로 대체)
  • OAuth 제공자 정보 — Google 계정 식별용
✅

비밀번호·전화번호·주소·카드번호 등 민감 정보는 수집하지 않습니다.

수입·지출 내역, 예산, 자산 정보 등 가계부 데이터는 로그인한 계정에 연결되어 저장됩니다.

Google 계정 없이 로컬 모드로도 사용 가능합니다. 이 경우 어떠한 개인정보도 수집되지 않습니다.

2 데이터 저장 및 백업 방식

체크메이트는 가계부 데이터를 사용자 본인의 Google Drive 앱 전용 폴더에 백업합니다. Supabase는 로그인 인증에만 사용되며, 가계부 데이터를 저장하지 않습니다.

📱
데이터 입력 → 기기(브라우저 로컬 스토리지)에 즉시 저장
↓ 동기화 버튼 클릭 시
📂
본인의 Google Drive 앱 전용 폴더에 백업 (checkmate_backup.json)
↓ 앱 실행 시 자동 비교
🔄
기기 ↔ Drive 중 더 최신 데이터로 자동 선택
🔐

Drive 앱 전용 폴더는 본인만 접근 가능합니다. 개발자를 포함한 누구도 해당 데이터에 접근할 수 없습니다.

ℹ️

앱 최초 실행 시 Drive 백업 동의 여부를 선택합니다. 동의하지 않으면 이 기기 로컬에만 저장됩니다.

3 Google 계정 없이 사용하기 (로컬 모드)

Google 계정 없이도 앱을 사용할 수 있습니다. 이 경우 모든 데이터는 이 기기의 브라우저에만 저장됩니다.

  • 어떠한 개인정보도 수집·전송되지 않습니다
  • 다른 기기에서는 데이터를 볼 수 없습니다
  • 브라우저 캐시·사이트 데이터 삭제 시 데이터가 영구 삭제됩니다
  • 앱(브라우저) 삭제 또는 휴대폰 초기화 시 삭제됩니다
⚠️

로컬 모드는 데이터 유실 위험이 있습니다. 설정 탭의 JSON 백업을 정기적으로 권장합니다.

4 추가 로컬 백업 (JSON)

설정 탭 → 데이터 백업/복원에서 전체 데이터를 JSON 파일로 직접 내보낼 수 있습니다.

  • 내보낸 파일은 기기·드라이브 등 원하는 곳에 보관 가능
  • 파일을 불러오면 언제든 데이터 복원 가능
  • 계정 변경 또는 기기 이전 시 데이터 이관에 활용 가능
⚠️

JSON 파일에는 가계부 데이터 전체가 포함됩니다. 파일을 안전한 곳에 보관하세요.

5 제3자 제공 및 공유

체크메이트는 사용자의 데이터를 제3자에게 판매하거나 제공하지 않습니다.

  • 광고 네트워크 연동 — 없음
  • 분석 서비스(Google Analytics 등) 연동 — 없음
  • 마케팅 목적 데이터 활용 — 없음

아래 서비스를 이용하며, 해당 서비스의 개인정보처리방침이 함께 적용됩니다.

  • Google 개인정보처리방침 — 로그인 인증 및 Drive 백업
  • Supabase 개인정보처리방침 — 로그인 인증 처리

6 데이터 보관 및 삭제

가계부 데이터는 사용자의 Google Drive에 보관됩니다.

  • 로그아웃 시 — Drive 데이터 유지, 로컬 데이터 유지
  • 브라우저 캐시 삭제 시 — 로컬 데이터만 삭제, Drive 데이터 유지
  • Google Drive 앱 연결 해제 시 — Drive의 백업 파일 즉시 삭제
  • 계정 탈퇴 요청 시 — Supabase 계정 정보 즉시 삭제
✅

데이터 삭제 또는 계정 탈퇴를 원하시면 아래 이메일로 요청해 주세요. 즉시 처리해 드립니다.

7 인터넷 연결 및 앱 권한

체크메이트는 아래 목적으로만 인터넷 연결을 사용합니다.

  • Google 소셜 로그인 인증
  • Google Drive 앱 전용 폴더에 데이터 백업 및 복원
  • 앱 리소스(폰트 등) 로드 및 캐시

카메라, 마이크, 위치, 연락처 등 민감 권한은 요청하지 않습니다.

8 아동 개인정보 보호

체크메이트는 만 14세 미만 아동을 대상으로 하지 않으며, 아동으로부터 고의로 개인정보를 수집하지 않습니다. 만 14세 미만 아동이 로그인한 사실을 인지한 경우 해당 계정의 데이터를 즉시 삭제합니다.

9 방침 변경 안내

개인정보처리방침이 변경될 경우 앱 내 공지 또는 본 페이지를 통해 사전에 안내드립니다. 변경된 방침은 공지 후 7일이 경과한 시점부터 효력이 발생합니다.

지속적인 앱 사용은 변경된 방침에 동의하는 것으로 간주됩니다.
