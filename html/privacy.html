1 수집하는 개인정보

체크메이트는 어떤 개인정보도 수집하거나 서버로 전송하지 않습니다.

  • 로그인 계정 없이 안전하게 이용합니다
  • 이름·이메일·전화번호 등 식별 정보 수집 없음
  • 비밀번호·카드번호·계좌번호 등 민감 정보 수집 없음
  • 위치·광고 식별자 등 행태 정보 수집 없음
✅

수집하지 않는 정보는 유출되지 않습니다.

2 데이터 저장 방식

수입·지출 내역, 예산, 자산 등 모든 가계부 데이터는 사용자 기기의 브라우저 로컬스토리지에만 저장됩니다.

  • 서버로 전송되거나 외부에 저장되지 않습니다
  • 다른 기기와 자동 동기화되지 않습니다
  • 개발자를 포함해 누구도 데이터에 접근할 수 없습니다
⚠️

로컬 저장 특성상 브라우저 캐시 삭제·앱 재설치·기기 이전 시 데이터가 손실될 수 있습니다. 아래 §3 백업 기능을 활용해주세요.

3 데이터 백업 (JSON)

설정 탭 → 데이터 백업/복원에서 전체 데이터를 JSON 파일로 직접 내보낼 수 있습니다.

  • 내보낸 파일은 기기·공식 클라우드 등 원하는 곳에 직접 보관합니다
  • 파일을 불러오면 언제든 데이터를 복원할 수 있습니다
  • 기기 이전·브라우저 변경 시에도 동일한 방식으로 이관 가능합니다
⚠️

JSON 파일에는 가계부 데이터 전체가 포함됩니다. 안전한 곳에 보관해주세요.

4 제3자 제공 및 외부 서비스

체크메이트는 제3자 서비스를 일체 사용하지 않습니다.

  • 광고 네트워크 연동 — 없음
  • 분석 서비스(Google Analytics 등) 연동 — 없음
  • 외부 인증·클라우드 동기화 — 없음
  • 데이터 판매·공유 — 없음

앱 로딩 시에는 웹 상에서 공개적으로 제공되는 웹폰트(나눔스퀘어네오) 자원만 다운로드하며, 이 과정에서는 개인 식별 정보가 전송되지 않습니다.

5 데이터 삭제 및 관리

모든 데이터는 사용자 본인의 기기에만 있으므로, 아래 방식으로 직접 제거하실 수 있습니다.

  • 설정 탭 → 데이터 초기화 (앞으로 추가 예정)
  • 브라우저 설정 → 사이트 데이터 삭제
  • 앱(PWA) 삭제 또는 브라우저 재설치
✅

서버에 저장된 데이터가 없으므로 별도 탈퇴 절차가 필요 없습니다.

6 인터넷 연결 및 앱 권한

체크메이트는 아래 목적으로만 인터넷 연결을 사용합니다.

  • 앱 리소스(웹폰트 등) 최초 로드 및 캐시

카메라·마이크·위치·연락처 등 민감 권한은 요청하지 않습니다. 설치 후에는 인터넷 연결 없이도 완전하게 사용 가능합니다.

7 방침 변경 안내

개인정보처리방침이 변경될 경우 앱 내 공지 또는 본 페이지를 통해 사전에 안내드립니다. 변경된 방침은 공지 후 7일이 경과한 시점부터 효력이 발생합니다.
